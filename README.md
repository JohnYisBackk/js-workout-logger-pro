## 📖 About

Workout Logger PRO is a fitness tracking application built with HTML, CSS, and JavaScript.

The app allows users to log workouts, track exercises, monitor training progress, and manage workout history through a clean and responsive interface.

All workout data is stored locally using Local Storage, allowing users to keep their progress even after refreshing the page.

---

## ✨ Features

* Add new workout sessions
* Track exercise names
* Log sets and repetitions
* Workout history management
* Delete workouts
* Search functionality
* Filter workouts
* Progress statistics
* Local Storage persistence
* Responsive design

---

## 🛠 Technologies Used

* JavaScript (ES6)
